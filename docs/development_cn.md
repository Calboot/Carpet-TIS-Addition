# 开发

当前主开发 MC 版本：**1.15.2** (与 `versions/mainProject` 相同)

本模组借助 [preprocessor](https://github.com/ReplayMod/preprocessor)，在单一分支下对所有受支持的 MC 版本进行维护

常见的分支的用途:

- `master`: 稳定，发布版本，也是 PR 的首选目标
- `dev`: 特性开发
- `future`: 快照适配

master 分支通常仅接受文档更新

除非必要，尽量不要影响版本兼容性

英文文档与中文文档是逐行对应的
